## 🔥 What I'm Building

### 📱 TheBooker — React Native Mobile Platform
A booking platform with both booker-facing and business-facing sides, currently undergoing a full **New Architecture (Fabric) migration**.

- React Native 0.79 → 0.83+ New Architecture adoption
- Performance-critical crash debugging at the native/JS bridge level
- iOS & Android platform-specific issue resolution

### 🖥️ SatiScore — Web Dashboard Platform
Separate admin and client dashboards built with React + TypeScript + Vite.

- Production-grade Axios client with JWT auth, refresh token queue, and request cancellation
- TanStack React Query with custom retry logic and global error handling
- Dockerized with multi-stage builds — slashed image size from ~1.7GB down to **~50–80MB**
